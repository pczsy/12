    • VS code 快捷键的应用。
    • 目标：逐步记忆用到的快捷键

     

    1. 快速生成html5的结构: 英文格式下: ! + 空格
    2. 保存：CTRL+S
    3. 打开文件： CTRL+O
    4. 添加注释: CTRL+ / 或者先CTRL+K，再按CTRL+C      
    5. 取消注释： CTRL+/ 或者先CTRL+K，再CTRL+U
    6. 撤销刚才的操作：Ctrl+Z 
      恢复刚才的操作：Ctrl+Shift+Z
    7. 生成多个相同的标签，加上*号。比如 div*3+回车，即生成3个div标签
    8. 生成父子级关系的标签，可以用 >号。比如 ul>li +回车 
    9. 以上7/8可以合起来用。比如ul>li*3，即生成ul和3个li的标签
    10. .demo/.vip/#vip/input:text/input:radio 加回车，分别可得如下：
