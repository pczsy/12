    1、第一步安装node.js

    https://nodejs.org/en/

    前期可以下载软件包，后期熟练以后可以通过nvm进行 Node的版本切换以及升级

    然后window+r  输入cmd 打开命令行  输入node -v  查看版本号 如：

    2 、安装cnpm（淘宝国内加速）
    npm install -g cnpm --registry=https://registry.npm.taobao.org

    3、全局安装vue

    cnpm install vue-cli -g                //安装完成后在命令行输入 vue -V 查看当前的版本

    4 、创建项目
    vue init webpack  app          //webpack 后面跟的是项目创建的名称 自己随便定义-----（只要不是大写的子母、或者是中文）

    5、根据命令行提示下操作

    cd app 进入app目录

    6、npm run dev          //就启动了

     

    这样就可以书写让自己愉快代码了，像在自己定义的组件里做一些让你觉得优雅的事情

     

     

    /------注释-------/

    在项目开发过程中会遇到跨域，设置跨域代理就ok了  在config文件夹里的  index.js 里操作   //也可以使用 Nginx  做代理，需要有基础的哦

    proxyTable: {
    '/api': {
    　　target:'http://api.douban.com/v2', // 你请求的第三方接口
           changeOrigin:true, // 在本地会创建一个虚拟服务端，然后发送请求的数据，并同时接收请求的数据，这样客户端和服务端进行数据的交互就不会有跨域问题
           pathRewrite:{ // 路径重写，
               '^/api': ''    /---因为在最初的域名上是没有/api的，所以路径重写将它替换为空---/ 
              }
    }
