    步骤1  Frameset 布局

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>视频网站大杂烩</title>
    <link rel="stylesheet" href="">
</head>
<frameset cols="200px ,*">
    <frame src="html/list.html" noresize="noresize"/>
    <frame name="video" src="html/video.html" noresize="noresize"/>
</frameset>
</html>

    css要求：左边导航栏列表居中，颜色，字体大小

    *{
    margin:0px;
    padding: 0px;
}

    background-color: #f1f1f1;
}

    /* 消除有序列表的项目符号 list-style-type:none; */

    width: 100%;
    height: 40px;
    line-height:40px;;
