    图像标签 img
    属性  说明 值
    src 图片地址  url
    width 宽度  px %
    height  高度  px
    alt 图片未加载或加载失败的提示 提示文字

    有序列表 ol
    属性 说明 值
    type 符号类型 1(阿拉伯数) a,A(大小写字母) i,I(大小写罗马数)
    start 从第几个开始 数值 num 

     

     

     

     

     

     

     

     

    热区 area 
    属性 说明  值
    shape 热区形状  circle(圆)、rect(矩形)、poly(多边形)
    coords 坐标 坐标值
    href 地址 URL
    target 打开方式  _seft、_blank(新页面打开)

     

     

     

     

     

     

     

     

     

     

     

    表格 table tr(行) td(单元格)
    标签  属性内容和边框之间的距离  值
    caption 表格标题  
    border 边框宽度  px
    cellpadding 内容和边框之间的距离 px
    cellspacing  边框与边框之间的距离 px
    align 对齐方式 center left right
    valign 垂直对齐方式 top bottom middle
    colspan 合并列单元格  数字
    rowspan 合并行单元格 数字
    bgcolor  背景颜色 颜色单词

     

      

     

     

     

     

     

     

     

     

     

     

     

     

     

    表单 form 
    属性  说明 值
    name 表单名称   
    action 提交地址 处理表单信息的地址
    method 提交方式 get(地址栏)、post
    target 打开方式 _seft _blank
    enctype MIME类型  text/plain(默认)(多用途互联网扩展类型) multipart/form-data

     

     

     

     

     

     

     

      

     

     

     

     

     

     

     input
    属性  说明 值

    value 值  
    size 长度(默认20) number
    readonly 只读  
    maxlength 允许输入最大字符  
    minlength 允许输入最小字符  
    required 必填  
    disabled 禁用  
    placeholder 占位符(提示信息)
