    • 纯css更改图片颜色的技巧
    原文出处:本文由博客园博主张小中提供。
    原文连接:https://www.cnblogs.com/zjz666/p/11124872.html

    tips： JPG、PNG、GIF 都可以，但是有一个前提要求，就是黑色纯色，背景白色

    .pic1 {
        background-image: url($img), linear-gradient(#f00, #f00);
        background-blend-mode: lighten;
        background-size: cover;
    }
