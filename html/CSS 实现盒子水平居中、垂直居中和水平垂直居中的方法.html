    方法一：使用margin: 0 auto;（只适用于子盒子有宽的时候）

    方法二：text-align + display（子盒子有或没有宽度的时候都适用）

    方法三：position 定位（只适用于子盒子有宽度和高度的时候）

    方法四：position + transform（子盒子有或没有宽高的时候都适用）

    方法五：flex 布局（弹性布局）（子盒子有或没有宽高的时候都适用）
