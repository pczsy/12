    使用

    拿pulse为例子

    导入上述的方法

    全是再样式中操作

    @keyframes pulse {
    25% {
        transform: scale(1.1);
    }

    75% {
        transform: scale(.9);
    }
}

    配置方法

    .pulse:hover {
    animation-name: pulse;  #方法名称
    animation-duration: 1s; #方法时间
    animation-timing-function: linear; #从开头到结尾以相同的速度来播放动画
    animation-iteration-count: infinite; #动画的次数
}
