    border 边框css属性
    边框颜色 border-color
    边框样式 border-style：solid （实线）dashed（虚线）默认为none
    边框粗细 border-width：1px；默认是3px
    border的简写 border：border-width；border-style；border-color；


    div 块 盒子
    1、就是标签名 没有什么特殊的标签属性
    2、主要用来排版布局
    3、宽度是100%，谷歌浏览器默认有8像素的外边距，清除这8px 用
    body{
    marign：0；
    }

    4、这种天生占用宽度为100%的标签我们把他叫做块级元素；
    比如 div、h1、h6等等
    特点：独占一行，可以设置宽高。
    5、大小宽高不是100%，是按照内容的多少决定大小的，这种标签叫做内敛元素（行级元素）
    比如 a span标签
    特点 不独占一行，不可以设置宽高。
    块级元素div分析
    1、外边距 marign
    2、内边距 padding
    3、边框 border
