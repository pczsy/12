    position有四种选择：

    1. static，默认，没有定位
    2. relative 相对位置
    3. absolute 绝对位置
    4. fixed 固定位置（一开始我觉得不好用，但是试了几次后发现还是很适合自己的审美的，具体效果的话参考在网站两边跟着你的广告，如：）

    相对位置相对原来位置进行偏移，设置后的元素仍在标准流，对其他元素位置没有任何影响。

    绝对定位以祖先元素或浏览器窗口（没有祖先时）为基准偏移，设置后的元素从标准流中脱离，对其他元素有一定的影响（有需求时可以借用此特性让元素脱离标准流而在原来位置）

    补充：z-index可以设置元素的堆叠顺序
