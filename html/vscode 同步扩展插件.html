    同步 VSCode 配置

    打开 VSCode，安装 settings sync 插件，然后按 shift+alt +u（下载配置的快捷键），然后根据提示输入 GitHub token 和 GitHub gist，稍等片刻就可以看到配置从 GitHub 同步完成啦～
