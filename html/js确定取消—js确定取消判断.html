    国瑞前端： js确定取消，在html界面中，有css模拟的模态框，这样显示的就会更好看一些，那么javascript有没有自带的弹框呢，当然是有的，接下来我就来给大家介绍一下把：

    js确定取消-警告框alert()

    起到警告作用

    js确定取消-确认框confirm()

    <script charset='utf-8' type="text/javascript">
    document.onclick=()=>{
          if(confirm("你喜欢玫瑰花么?")){
            //条件成立就为真
            alert("true");

          }else{
            //条件不成立则为假
            alert("false");
          }
    }
    </script>

            //无输入则为假

          }
