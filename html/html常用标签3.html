    表单内容：
    form 表单
    input 输入框
    name属性：标签的名字
    value属性：设置输入框的默认内容
    checked属性：设置单选或多选的默认项
    type属性：用于确定输入框的类型
    type属性的属性值：
    text:单行文本
    password：密码
    email：邮箱
    radio：单选
    checkbox：多选
    reset：重置
    submit：提交
    file：文件上传
    image：以图片作为提交按钮 同时需要设置 src属性 用于获取图片的路径
    button：普通按钮 需要使用value属性添加按钮上的默认文字
    placeholder属性：用户输入框的默认提示信息

    button标签 表示按钮 在form表单中相当于一个提交按钮

    下拉菜单：
    select 代表整个下拉菜单
    option 表示每一个下拉选项
    selected属性 表示设置默认显示的下拉选项

    文本域：
    textarea 代表一个文本域
    cols属性 代表一行最多可以设置多少个字符
    rows属性 代表一共可以显示多少行
    disabled属性 代表禁用文本域

     

    表格： 必须是工整的，就是一个完整的表格必须是一个规整的矩形
    table标签 表示一个表格
    border属性 表示设置单元格的边框
    cellpadding属性 表示单元格内容与边框之间的距离
    cellspacing属性 表示单元格与单元格之间的距离


    style="border-collapse: collapse" 代表合并单元格之间的边框线

    tr标签 代表表格的 行
    td标签 代表每一个单元格

    table标签
    属性：
    border 设置边框
    width 设置整个表格的宽度
    style="border-col0lapse: collapse" 设置合并边框线
    height 设置高度
    tr标签 表格的行
    td标签 表格的单元格
    属性 colspan 代表跨列 如果设置跨列，那么不管是跨几列，对应的当前行都要相应的减少对应的列数
    rowspan 代表跨行 如果设置跨行，那么跨几行，对应的后面的行就要相应减少一个单元格
    例如 rowspan="3" 那么后两行就要依次减少一个单元格
    thead标签 表格的表头 tr表示行 th表示表头的单元格
    tbody标签 表格的主体 tr表示行 td表示主体的单元格
    属性：
    align 表示单元内的文字水平对齐方式 left 左对齐 center 居中 right 右对齐


    caption标签 表格的标题
