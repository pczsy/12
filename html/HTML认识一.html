    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <title>Title</title>
    </head>
    <body>
    <!--注释内容-->
    </body>
    </html>

    标签的意义：

    1、<!DOCTYPE html>   指明这是HTML5的规范

    2、<html lang="en"></html>  一对html标签表示整个网页

    　　2.1、<html lang="en">    lang="en"     en表示以英文方式处理

    3、<head></head>  一对head标签是整个网页的头部

    　　3.1、<meta charset="UTF-8">   编码格式以utf-8方式

    　　3.2、 <title>Title</title>    title是指标题   

    4、<body></body>  body标签是整个网页的身体，真正显示内容的地方

    　　4.1、<!--注释内容-->      快捷键 ctrl + l  快速注释
