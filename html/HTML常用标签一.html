    html文本格式化标签

    1. 在网页中，有时需要为文字设置粗体 、斜体 或下划线 效果，这是就需要用到HTML中的文本格式标签，是文字以特殊的方式显示
    2. 标签语义：突出重要性，比普通文字更重要
    语义 标签 说明
    加粗 <strong></strong>或者<b></b> 推荐使用<strong>标签，语义更加强烈
    倾斜 <em></em>或者<i></i> 推荐使用<em>标签，语义更加强烈
    删除线 <del></del>或者<s></s> 推荐使用<del>标签，语义更加强烈
    下划线 <ins></ins>或者<u></u> 推荐使用<ins>标签，语义更加强烈 
    <!DOCTYPE html>
<html lang="en">
    <head>
        <mate charset="UTF-8">
        <title></title>
    </head>
    <body>
        <strong >我是加粗</strong>
        <b>我也是加粗</b>
        
        <em>我是斜体</em>
        <i>我是斜体</i>
        
        <del>删除线</del>
        <s>删除线</s>
        
        <ins>下划线</ins>
        <u>下划线</u>
    </body>
</html>

    <div>和<span> 标签

    1. <div> 和<span> 没有语义，他们就是一个盒子，用来装内容的。
    <!DOCTYPE html>
<html lang="en">
    <head>
        <mate charset="UTF-8">
        <title></title>
    </head>
    <body>
        <div>这是div</div>
        <span>这是span</span>
    </body>
</html>
    1. div是division的缩写，表示分割，分区
      • 特点：一行只能放一个div
    2. span以为跨度，跨距
      • span一行可以放多个<span>

    图像标签和路径

    1. 图像标签
      • 在HTML标签中，<img> 标签用于定义HTML页面中的图像
      • 单词image的缩写，以为图像
      • src是<img> 标签的必须属性，他用于指定图像文件的路径和文件名
    <img src="图像URL" />
    属性 属性值 说明
    src 图片路径 必须属性
    alt 文本 替换文本，图像不能显示的文字
    title 文本 提示文本，鼠标放在图像上，显示文字
    width 像素 设置图像的宽度
    height 像素 设置像素的高度
    border 像素 设置图像的边框粗细
