    • Flutter 安装笔记

    原文连接:https://www.cnblogs.com/CharlesJian/p/11045211.html

    一. 安装镜像(有vpn的不用理)
    1  打开终端 输入 open ~  ，回车
    2  双击 .bash_profile 
    3  添加以下代码 后保存关闭即可（代码可能会变请直接到https://flutter.dev/community/china 复制）
         export PUB_HOSTED_URL=https://pub.flutter-io.cn
       export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn    
     
    二. 获取flutter SDK 并安装体验
    1 直接到 https://flutter.dev/docs/development/tools/sdk/archive?tab=macos 下载最新版并解压
    2 解压解压包到本地文件夹即可
    3 添加flutter 开发工具到 path 中 (打开终端 输入
     open ~  
    找 .bash_profile )
      然后添加
     export PATH=/Users/luby/Documents/flutter/bin:$PATH  
      PS: /Users/luby/Documents/flutter/bin 是你的刚解压的flutter SDK 下的bin 路径
    4 至此 flutter 安装环境就OK了，你可以在终端 输入 flutter 检验是否安装成功
     
    5  flutter doctor 输入这个可检查你其他配套设施是否已安装
     
     
    # Android 环境变量
    export ANDROID_HOME=/Users/luby/Library/Android/sdk
    export PATH=/Users/luby/Documents/flutter/bin:$PATH 
    export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.aliyun.com/homebrew/homebrew-bottles
    三 安装xcode(苹果开发编辑器)
    1 直接到 https://developer.apple.com/xcode/ 下载 或者打开app store
    2 配置xcode 命令行工具以使用新安装的xcode版本
       终端:
     sudo xcode-select --switch  /Applications/Xcode.app/Contents/Developer
    3 终端输入许可协议 :  
    sudo xcodebuild -license 
    输入后一顿空格并回车就行
    4 设置打开 IOS模拟器 
       终端: 
    open -a Simulator
    四 Android开发环境设置和 flutter插件安装
    1 直接到 https://developer.android.google.cn/studio下载安装包
    2 添加设备： Tools ->  AVD Manager  安装对应手机版本即可
    3 添加flutter 插件  Plugins -> Browse repositories -> 搜索Flutter并下载(估计还要下载dart)
    4 安装完毕重启 android studio 即可运行你的flutter APP拉
    五 终端创建 flutter app 并试运行 
     1    输入 
    cd Desktop/
     2    然后
     flutter create my_app   // (项目名字)
     
      cd my_app 
     
      flutter run
