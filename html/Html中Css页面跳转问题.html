    没有指定<a>的target属性值时，默认是"_blank,可以参考以下代码的设置来控制
    <a href="#" target="_blank">链接</a>

    target的值
    target="_blank":在新窗口中浏览新的页面。
    target="_self":在同一个窗口打开新的页面。
    target="_parent":在父窗口中打开新的页面。（页面中使用框架才有用）
    target="_top" :以整个浏览器作为窗口显示新页面。（突破了页面框架的限制）
