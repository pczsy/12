    • 实现单选框点击label标记中的文字也能选中
    原文出处:本文由博客园博主xhshen提供。
    原文连接:https://www.cnblogs.com/sxh24/p/htmls.html

    实例：

    <label for="man"> 
    <input type="radio" value="男" name="man" id="man"/> 
    男 
    </label> 
    <label for="nv"> 
    <input type="radio" value="女" name="man" id="nv"/> 
    女 
    </label> 

    <label> 标签的 for 属性应当与相关元素的 id 属性相同 。

    <label> 标签为 input 元素定义标注（标记）。label 元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。当点击 label 元素内的文本时，就会触发此控件。从而达到点击label中的文字也可以选中的效果。
