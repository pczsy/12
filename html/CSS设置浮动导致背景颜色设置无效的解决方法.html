    • CSS设置浮动导致背景颜色设置无效的解决方法
    原文出处:本文由博客园博主DoubleL2提供。
    原文连接:https://www.cnblogs.com/gsdoublel/p/11260564.html

    float浮动会使父元素高度塌陷，父级元素不能被撑开，所以导致背景颜色不能被撑开

    解决方法：

    1. 对父元素设置高度
    2. 对父元素设置 overflow：hidden清除浮动
    3. 把父元素也设置为float浮动
    4. 结束标签前加代码：<div style="clear:both"></div>
