    Xtreme Admin Dashboard 是基于 bootstrap 4 的管理仪表板和控制管理面板。 Xtreme 是完全响应的 HTML 模板，基于 CSS 框架 Bootstrap 4 并基于模块化设计。它在设计中利用了所有 Bootstrap 组件，并重新设计了许多常用插件，以创建一致的设计，可用作后端应用程序的用户界面。Xtreme Admin Template 基于模块化设计，可以轻松定制和构建。
