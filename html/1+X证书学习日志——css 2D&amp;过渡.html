    css 位移常用属性

                transform:translate(x,y)；
            transform:translateX();
            transform:translateY();

    旋转属性

                2d旋转：
            transform:rotate()；  /*绕着中心点转动*/
            transform:rotateX()； /*绕着X轴转动*/
            transform:rotateY()； /*绕着Y轴转动*/


        变形原点：
            transform-origin: X轴位置  Y轴位置；

    缩放属性

                   transform:scale(); /* 0~1为缩小，1以上为放大 */

    过渡属性：

                transtion:all 时间 延迟时间 动画的类型(匀速linear);
